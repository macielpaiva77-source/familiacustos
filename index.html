👨‍👩‍👦 Controle de Custos Família
Configure a sincronização via GitHub — feito uma vez, funciona em todos os dispositivos.
  1. No GitHub, clique na sua foto de perfil → Settings → Developer settings (no final do menu esquerdo).
  2. Clique em Personal access tokens → Tokens (classic) → Generate new token (classic).
  3. Em Note escreva custos-familia. Em Expiration selecione No expiration.
  4. Marque a caixinha ✅ repo (a primeira do grupo repo). Role para baixo e clique em Generate token.
  5. Copie o token gerado (começa com ghp_) e cole abaixo. Só aparece uma vez!
🔒 O token é salvo apenas no seu navegador (localStorage). Nunca é enviado para terceiros — a comunicação é direta entre seu navegador e a API do GitHub. Para revogar o acesso a qualquer momento, basta deletar o token nas configurações do GitHub.
