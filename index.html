🌐 Como Hospedar

Acesse esta planilha de qualquer lugar, sem custo

✅ 100% GRATUITO

☁️ Netlify Drop

A forma mais rápida e simples. Arraste o arquivo HTML para o site e ele fica online em segundos, com HTTPS incluso e um link para acessar de qualquer dispositivo.

  • Acesse app.netlify.com/drop
  • Arraste o arquivo familia.html para a área indicada
  • Pronto! Você recebe um link como abc-xyz.netlify.app
  • Para atualizar: faça novo deploy arrastando o arquivo novamente
⚠️ Atenção: Os dados ficam salvos no localStorage do navegador, não na nuvem. Para sincronizar entre dispositivos, use a opção do servidor em casa abaixo.
✅ GRATUITO

📄 GitHub Pages

Hospedagem permanente e gratuita pelo GitHub. Ideal se você quiser versionar e manter histórico de alterações no arquivo.

  • Crie uma conta em github.com
  • Crie um repositório público chamado familia-custos
  • Faça upload do arquivo como index.html
  • Vá em Settings → Pages → Branch: main → Save
  • Link: seuusuario.github.io/familia-custos

🏠 CASA Servidor no seu Computador + IP Público

Esta é a opção mais completa para uso doméstico: o arquivo fica no seu PC, você acessa de qualquer lugar via internet, e os dados podem ser centralizados num único lugar acessível por todos os dispositivos da família.

PASSO 1 — Iniciar servidor

Com Python instalado, abra o terminal na pasta do arquivo:

# Python 3 (recomendado) python -m http.server 8080 # Se for Python 2 python -m SimpleHTTPServer 8080

Acesse localmente em: http://localhost:8080

PASSO 2 — Acesso externo grátis (ngrok)

O ngrok cria um túnel público sem precisar mexer no roteador:

# 1. Baixe em ngrok.com (gratuito) # 2. Crie conta e pegue o token # 3. Configure: ngrok config add-authtoken SEU_TOKEN # 4. Exponha a porta: ngrok http 8080

Link gerado como: abc123.ngrok.io

PASSO 2 (alternativo) — DDNS + Port Forward

Para um link fixo e permanente sem depender do ngrok:

  • Crie conta no DuckDNS.org (grátis)
  • Crie um domínio: minhaFamilia.duckdns.org
  • No roteador, habilite Port Forwarding: porta 8080 → IP do seu PC
  • Instale o DuckDNS client no PC para atualizar o IP automaticamente
  • Acesse de qualquer lugar pelo seu domínio

📱 Dica: Acesso pelo Celular

Se o servidor estiver rodando no PC e você estiver na mesma rede Wi-Fi de casa, acesse pelo celular usando o IP local do PC:

# Descubra o IP local do seu PC: # Windows: ipconfig # Linux/Mac: ifconfig ou ip addr # Acesse no celular: http://192.168.1.XXX:8080

🔄 Sincronizar Dados Entre Dispositivos

O localStorage é local por dispositivo. Para compartilhar entre PC, celular e tablet da família, a solução é acessar sempre pelo mesmo endereço (servidor em casa ou Netlify) e usar um único dispositivo como "fonte da verdade".

Opção alternativa: use o botão "Exportar CSV" no Dashboard para backup, e futuramente posso evoluir o app para salvar em Google Sheets ou Supabase (gratuito).
